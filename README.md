# SAT 求解 迷宫问题

## 问题描述

用SAT 的方法, 求在给定的n × n 的迷宫中, 求以(1, 1) 为起点,(n, n) 为终点的一条简单路(不含环).

1) 请给出编码的方法.
2) 通过利用开源SAT 求解器, 给出附件例子的结果, 编码源代码和相关的CNF 文件.

例子格式说明：
第一行一个数n，表示迷宫的大小为n × n。
接下来一个n × n 的矩阵，0 代表无障碍，1 代表有障碍。
